From January 2019 to the Summer of 2021 five web applications were redesigned for the Ministry of Health. In order to achieve a visually consistent experience the BC Government Design System was used. The need arose for components not currently in the system which led the team to reach out to the system maintainers at the Government Digital Experience Division (GDX). We ended up collaborating and adding new components together. We also made a graphics file containing all design system components so that they could be quickly and easily used on new projects.

TYPOGRAPHY

GDX chose BC Sans as the standard typeface for all BC Government websites. This happened mid-project so we promptly switched over to it and updated the styleguide.

GRID

At project start the design system didn’t have a standard grid so we established one and added to the styleguide.

COLOUR

Standard colours were used throughout all web applications designed for the Ministry of Health.

ICONS

Font Awesome is the standard icon set for the BC Government Design System. Commonly used icons are kept in the styleguide for quick access.
